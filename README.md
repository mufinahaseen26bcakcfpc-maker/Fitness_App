# 🏋️ PULSE - Fitness App

## 📌 Project Overview
PULSE is a fitness application designed to help users maintain a healthy lifestyle. The app provides workout guidance, nutrition support, health monitoring, and progress tracking in a simple and user-friendly interface.

## 🎯 Features
- 💪 Strength Training
- 👥 Group Workout
- ❤️ Heart Rate Monitoring
- 🥗 Healthy Nutrition
- 📱 Simple & Attractive UI
- 📈 Fitness Progress Tracking

## 🎨 Color Palette
- 🔵 Blue – Primary Color
- 🟠 Orange – Secondary Color
- 🟢 Green – Accent Color
- ⚫ Black – Text Color

## 🛠️ Tools Used
- Figma
- UI/UX Design Principles

## 👥 Target Users
- Fitness Beginners
- Gym Enthusiasts
- Health-Conscious People

## 💡 Motivation
**"Your Fitness. Your Future."**

*Track your progress, stay consistent, and become the strongest version of yourself.*
